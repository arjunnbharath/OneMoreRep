# Workout pictures by muscle

Add images for each muscle group in its folder. They are served from `/images/workout-pic/`.

| Folder | Muscle |
|--------|--------|
| `chest/` | Chest |
| `back/` | Back |
| `shoulders/` | Shoulders |
| `biceps/` | Biceps |
| `triceps/` | Triceps |
| `abdominals/` | Abdominals |
| `legs/` | Legs |
| `calves/` | Calves |

## Suggested files

- **Muscle tile (Home grid):** e.g. `chest/cover.jpg` → `/images/workout-pic/chest/cover.jpg`
- **Per exercise:** e.g. `chest/barbell-bench-press.jpg`

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
